## Parte B 
1. Un algoritmo son pasos ordenados que se siguen para resolver un problema. Esto se considera conocimiento imperativo. Algunos algoritmos utiles empleados en programación es Brute Force, Divide and Conquer, etc. 
2. El print() es el output del código mientras que el return() se utiliza en una función para guardar y reutilizar un valor en alguna parte del código. El print() no guarda valores, solo las despliega en la pantalla. El print() lo utilizaría para depliegar un mensaje mientras que el return para guardar algún cálculo de una función. 
3. Scope es el alcance donde una variable existe y se puede usar. La variable local se define dentro de una función y solo se usa aquí. Una variable global se crea fuera de la funcion y puede usarse en todas partes del programa. 
4. Una lista es un conjunto de datos string o integers que se guardan en una sola variable. Se puede agregar elementos con append y recorrer la lista con bucles como for. 
Ejemplo: 
frutas = []

frutas.append("manzana")
frutas.append("pera")

for fruta in frutas:
    print(fruta)
Aquí se esta creando una lista y se utiliza frutas.append para agregar elementos a la lista. Luego, con un enlace "for" las imprimo. 
5. El método __init__ inicializa un objeto a partir de una clase. Es el método constructor y es lo que le da "vida" a un objeto a partir de los parámetros establecidos. Self se utiliza para hacer referencia al objeto actual y darle atributos tales como self.name, self.age, etc.
